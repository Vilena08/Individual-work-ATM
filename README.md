# Individual-work-ATM

Простая симуляция банкомата на C#.

## Что делает программа

- Вход по номеру счёта и PIN-коду
- Проверка баланса
- Внесение денег
- Снятие денег
- Перевод на другой счёт
- Выход из системы

## Использованный паттерн: Singleton

### Цель применения

Обеспечить существование только одного экземпляра банкомата во всей программе.

### Какую задачу решает паттерн

В приложении не должно быть ситуации, когда создаётся второй объект банкомата с другими данными о счетах. Если бы экземпляров было несколько, балансы пользователей в разных частях программы могли бы различаться — это вызвало бы ошибки в логике снятия и перевода денег.
